# Oria

**Oria – Nudge your choices, never dictate your fate.**  
**When you need a light on your path, I’m here.**

**Oria —— 在你身邊的明燈。讓你看清前路。**

Oria is a personal guidance PWA that combines BaZi and MBTI to help users understand themselves more clearly and navigate relationships, work, and life decisions with gentle, contextual insight.

This product is designed to offer reflection and light guidance — not to dictate life choices or replace human judgment.

## Planned Stack
- Frontend: React + TypeScript + PWA + i18n
- Backend: Express + TypeScript
- Analysis microservice: FastAPI + Python
- Database/Auth: Supabase

## Build Strategy
This project will be built in phases.
The goal is to keep architecture understandable, reviewable, and under control.
